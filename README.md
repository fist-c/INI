# Contra Enjoyable
This is the source INI code of a freeware modification for C&C Generals based on Contra Classic mod.\
The mod repo contains in-dev INI files only for so-called Contra Classic "Patch".\
In other words: Contra 007 -> Contra Classic -> Contra Enjoyable

Here you can finally enjoy the complete domination and OP-ness of your faction over the enemies instead of trying to think and use your own brain to get victory.

The mod provides a player with totally new expirience of disbalanced gameplay, meanwhile staying the same and well-familiar Contra 007. Core idea of this whole project is to introduce more and more disbalancing features with a little amount of original updates. Visual part remains almost the same, keeping old-cuckold Contra style which reminds you of your non-binary nature after launching. Besides the shizo-fixes mod introduces a few new units and building, mostly stolen from other mods.
